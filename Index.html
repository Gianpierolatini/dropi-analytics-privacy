Política de Privacidad - Dropi Analytics

Última actualización: 30 de junio de 2026

1. Información que recopilamos

Dropi Analytics no recopila ni almacena datos personales de los usuarios. La extensión únicamente:

  • Lee el token de autenticación JWT desde el localStorage de app.dropi.cl (solo en el navegador local del usuario)
  • Guarda dicho token en el almacenamiento local de Chrome (chrome.storage.local) para evitar inicios de sesión repetidos

2. Uso de datos

Los datos obtenidos se usan exclusivamente para:

  • Consultar el catálogo de productos de la API oficial de Dropi (api.dropi.cl)
  • Mostrar información de productos, márgenes y órdenes dentro del popup de la extensión

No se transmite ningún dato a servidores de terceros ni al desarrollador de la extensión.

3. Datos que NO recopilamos

Esta extensión NO recopila: información de identificación personal, historial de navegación, datos financieros, comunicaciones personales, ubicación ni ningún otro dato sensible.

4. Almacenamiento local

El token JWT se almacena localmente en el dispositivo del usuario mediante chrome.storage.local. Este dato nunca sale del dispositivo y puede eliminarse desinstalando la extensión.

5. Permisos utilizados

  • storage: Para guardar el token de sesión localmente
  • activeTab: Para leer el token desde la página de Dropi
  • scripting: Para inyectar el script que obtiene el token de autenticación
  • api.dropi.cl: Para consultar el catálogo de productos
  • app.dropi.cl: Para obtener el token del usuario autenticado
